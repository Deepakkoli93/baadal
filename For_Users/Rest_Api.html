For Users Rest Api

Tuesday, March 22, 2016 11:58 PM

Usage of REST APIs

URI Description Arguments
/wm/baadal/list/json List all the VMs and their security domains HTTP GET, no other arguments
/wm/baadal/json Add or delete a security domain entry HTTP POST data (add doamin), DELETE data (delete domain)

Listing all the VMs and their security domains

curl http://<controller ip>:8080/wm/baadal/list/json

Adding a security domain

To add a security domain the user needs to provide and array of json objects. Each json object must have and ip field and a tag field.

curl -X POST -d '[{ "ip": "10.0.0.1", "tag": "5" }]' http://<controller ip>:8080/wm/baadal/setip2tag/json

Deleting a security domain

curl -X DELETE -d '[{ "ip": "10.0.0.1", "tag": "5" }]' http://<controller ip>:8080/wm/baadal/setip2tag/json
